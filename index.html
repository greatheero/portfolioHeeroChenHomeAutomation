ATEN VK Control System

ATEN VK control system has been debuted in early 2014 with its IDE, VK Mobile App Configurator. We interviewed the system integrator and client and collection their feedback. We extracted the pain point and came out the solution to improve. For example, one of the usability issue is the property pane wasted much real estate, the height of the 16:9 screen. It is so hard to development user interface for iPad App. Therefore, we brought it on the right side of the screen. Now, the VK developer can have the optimal space to design their excellent user interface.

Central control system
ATEN VK2100 is the core of the control system. It is connecting to any other devices in the environment and deligate every conversation among devices. It directly send commands to the target devices and respond to any specific changes of the environment.
Design | Development tool
ATEN VK6000 is the utility software integrated with the configuration of the device connection and the user interface design. All the system integrator do is to create the project, define the protocol between VK2100 and device, design user interface by drag and drop UI component. Even, programing is turned into code-block so that they could be added and reused in the project. Last, publish the project to VK2100. It's done.
Mobile controller App
iPad is the optimal controller for the office automation. The system integrator customized the mobile App user interface for their client's demand. During meeting, the end user can get all access to the devices around their office through VK control system mobile App. Besides, they can ask any PC to play the presentation slides and brower the slide preview page by page. One of our mission is to delivering the excellent UI in every App updates.
Maintainace utilities
Fast maintenance makes deep relationship between the system integrator and their client. When the trouble happened, the system integrator access to VK control system via its Web User Interface remotely. They can check the monitoring parameter and download the logs files to find out the where and when the problem comes. More, we provides Firmware Upgrade Tool to scan the ATEN device in LAN and can have advance configuration on the multiple device in the single operation. Nowadays, the lite central control management software promises reduce the cost of the daily operation
